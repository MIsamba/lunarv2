 Features
* django framework with defined models.
* restful api integrated - GET Request


# Download & Setup Instructions

* 1 - Clone project: git clone PROJECT URL
* 2 - cd backend
* 3 - Create virtual environment: virtualenv env
* 4 - Activate virtual environment: env\scripts\activate
* 5 - pip install -r requirements.txt
* 6 - python manage.py runserver
* 7 - run the given Url to the browser

# Install react modules
* 1 - cd frontend
* 2 - npm install
